# Python Set Operations Playground

- A beginner-friendly Python 3 project designed to practice core data structures
and lazy iteration using real examples.
This project focuses on lists, sets, intersections, differences, and generators (yield).

## Features

- Practice working with Python lists and sets

- Perform set operations:

- Intersection

- Difference

- Union

- Convert between lists and sets

- Use generator functions (yield) for memory-efficient iteration

- Simple, readable examples for learning purposes

## Concepts Covered

- Lists vs Sets

- Removing duplicates using sets

- Finding common elements between collections
